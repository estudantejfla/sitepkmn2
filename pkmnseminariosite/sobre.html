Sobre o Projeto

O Criar Time Pokémon é um site para montar times de Pokémon, Projeto desenvolvido por João Felipe Lopes

Objetivo

Praticar HTML5, CSS3 e JavaScript com foco em responsividade, interatividade (localStorage) e experiência do usuário.

Tecnologias

  • HTML5 + CSS3
  • JavaScript
  • PokeAPI
  • Hospedagem:Vercel
